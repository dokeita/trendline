# 要約プロンプトの改善（トピック分類と重要度スコア）

## 概要

summarize Lambda のプロンプトを改善し、単なる箇条書きではなくトピックごとにグルーピングされた構造的なレポートを生成する。

## 現状

- プロンプトは「主要なトレンドを箇条書きで要約」という簡素な指示のみ
- トピックの分類や盛り上がり度の定量的な表現がない

## 対応内容

- [ ] `summarize/index.py`: プロンプトを以下の方針で改善
  - トピックごとにグルーピングして出力するよう指示
  - 各トピックに関連ポスト数やエンゲージメント合計を付記
  - トピックの盛り上がり度順にソート
  - 各トピックに代表的な投稿を1〜2件引用
- [ ] 出力フォーマットの例をプロンプトに含める（few-shot）

## 期待効果

- 情報が構造化され、ユーザーが素早く全体像を把握できる
- 定量的な指標により、どのトピックが最も注目されているか一目で分かる

## 備考

- 001（public_metrics）と 002（著者情報）の対応後に実施するとより効果的
